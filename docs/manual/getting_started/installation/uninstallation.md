---
title: Uninstalling
---

To uninstall , manually remove the install and build directories:

::: tabs
::: group-tab
Linux

```bash
rm -rf cyclonedds
rm -rf <install-location>
```

:::

::: group-tab
macOS

```bash
rm -rf cyclonedds
rm -rf <install-location>
```

:::

::: group-tab
Windows

Navigate to your install location and remove the directory.
:::
:::
